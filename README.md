# Croven_Events
Application to organize Events


1. URL Parameter Categories with Examples

Event
schedule.php?category=event&q=lollapalooza
schedule.php?category=event&q=coachella

Venue
schedule.php?category=venue&q=madison+square+garden
schedule.php?category=venue&q=the+fillmore

City
schedule.php?category=city&q=chicago
schedule.php?category=city&q=new+york

State
schedule.php?category=state&q=illinois
schedule.php?category=state&q=california

Performer
schedule.php?category=performer&q=radiohead
schedule.php?category=performer&q=billie+eilish

Date — single day
schedule.php?category=date&start=2025-07-04
schedule.php?category=date&start=2025-08-15

Date — range
schedule.php?category=date&start=2025-06-01&end=2025-06-30
schedule.php?category=date&start=2025-12-26&end=2026-01-01
